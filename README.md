# rascar_hack

Проект, выполненный в рамках хакатона по анализу химических данных.  
Репозиторий содержит решение задачи классификации для пар молекул, представленных в формате SMILES.

## О проекте

В основе проекта лежат данные с двумя молекулярными структурами (`SMILES1` и `SMILES2`) и целевой переменной `result`.  
Основная цель — подготовить признаки на основе химических представлений молекул и построить baseline-модель для предсказания результата.

## Что реализовано

- загрузка и анализ обучающего и тестового датасетов
- работа с молекулярными структурами через **RDKit**
- подготовка химических признаков и дескрипторов
- baseline-модель для классификации
- генерация файла предсказаний для сабмита
